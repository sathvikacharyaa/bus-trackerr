# Shivamogga – Mangalore Bus Tracker

## About the project

- Tracks a regular Shivamogga → Manipal → Mangalore super express bus used by students and daily commuters.[web:9]
- Focuses on MIT/Manipal users boarding toward Udupi and Mangalore.
- Covers the major route sequence: Shivamogga, Thirthahalli, Agumbe, Hebri, Perdoor, Hiriyadka, MIT/Manipal, Udupi, and Mangalore.
- Uses day-of-week patterns to estimate when the bus should depart Manipal and reach Udupi and Mangalore.
- Visualises the route with a clear list of stops and indicates which segment the bus should currently be on based on the live clock.
- Shows simple punctuality insights (how often the bus is early, on time, or late) to set realistic expectations for passengers.
- Includes convenience actions such as:
  - “Add to Google Calendar” for creating events with the usual departure and arrival times, so users can set reminders.

## Intended use

This project is meant for students and regular commuters who rely on this specific bus and want a quick way to check typical timings and get a rough idea of where the bus should be along its route on any given day.
